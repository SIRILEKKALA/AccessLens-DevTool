# AccessLens-DevTool

**AccessLens-DevTool** is a Next.js-based accessibility tool designed to help users analyze images for:
- 🔤 **Text (OCR)** using Optical Character Recognition
- 🎨 **Contrast checking** for accessibility compliance (WCAG)
